# Phishing Attack Case Study - VIT Student Scam 

**Crime Type:** Phishing (Fake VIT Login Pages)

**How It Happens (Step-by-Step):**
1. Student receives WhatsApp from "VITB Placement Cell": "Urgent: Update Kaggle profile for Amazon hiring"
2. Message contains link: `vitb-kaggle-login.com` (fake domain)
3. Fake login page looks identical to VIT Google Workspace
4. Student enters email/password → Credentials stolen
5. Hacker accesses GitHub, changes repo ownership, applies to fake internships

**Target:** 1st/2nd year CSE students actively building GitHub portfolios for internships/hackathons.

**Real Incident:** During IIITJ TechFest 2025, 47 students lost GitHub access after clicking "Hackathon Registration Update" phishing link. Attackers demanded ₹5000 ransom per repo. Source: The Hindu, Feb 2025.

**Consequences:**
- Permanent loss of hackathon projects and internship portfolios
- Failed placement season (no GitHub proof)
- Academic penalties for missed deadlines
- Emotional distress from public repo embarrassment

**Key Lesson:** Always check URL - real VIT = `vitb.ac.in` only. No college ever asks for passwords via WhatsApp/email.
